**13 파일 정보 시스템 방식과 비교할 때 데이터베이스 시스템 방식이 갖는 장점은 무엇인지 설명하시오.**

**14 시대별 데이터베이스 시스템의 성장 과정을 개략적으로 설명하시오.**

**16 데이터베아스의 주요 특성 4가지를 설명하시오.**
1. 통합 데이터: 여러 소스에서 수집된 데이터를 통합하여 일관된 형태로 저장
2. 저장 데이터: 데이터를 장기간 보관하며 필요할 때마다 접근할 수 있는 방식으로 저장
3. 공유 데이터: 여러 사용자가 동시에 접근할 수 있으며, 다양한 애플리케이션에서도 데이터를 공유할 수 있다
4. 운영 데이터: 실제 운영에 필요한 데이터가 포함되며, 실시간으로 처리되고 활용되는 데이터를 의미한다
